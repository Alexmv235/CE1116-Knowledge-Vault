## 📚 Idea/Concepto 
El **Sprint Backlog** es el subconjunto de elementos del Product Backlog que el equipo selecciona para trabajar durante un sprint específico. Es el plan de trabajo para esa iteración; contiene las tareas que el equipo se compromete a terminar para cumplir con el objetivo del ciclo actual.
